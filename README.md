# FeelFill

FeelFill is a compact assistive device designed to support blind and visually impaired individuals in measuring liquid levels safely and independently. The system uses non-contact capacitive sensing to detect liquid levels through the container wall and provides real-time audio and vibration feedback.

---

## Project Overview

Traditional liquid measurement methods rely on visual cues, making them difficult for visually impaired users. FeelFill addresses this limitation by enabling accurate, contactless liquid level detection using embedded sensing and feedback systems.

The device is designed as a portable clip-on module that can be attached to common cups and containers. It processes sensor data in real time and provides progressive feedback as the liquid level rises.

---

## Key Features

- Non-contact capacitive liquid level detection  
- Real-time monitoring of liquid fill levels  
- Audio feedback using buzzer alerts  
- Vibration feedback for tactile guidance  
- Compact clip-on design compatible with multiple cup types  
- Rechargeable and portable operation  
- Low-power embedded system design  

---

## Working Principle

The system operates based on capacitive sensing. A sensor placed externally on the cup detects changes in capacitance caused by variations in the dielectric medium as the liquid level increases.

The microcontroller processes these changes and maps them to predefined threshold levels. Feedback is then provided to the user through vibration patterns and audio signals corresponding to different fill stages.
